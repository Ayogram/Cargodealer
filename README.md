# CargoDealerInc – VS Code Starter (Static Prototype)

## What this is
A static HTML/CSS/JS prototype that includes:
- Red Star-inspired hero + quick-service panel
- Tracking input (CDI-XXXXXX) with a 6-step timeline
- Services section using your provided images (Air / Ocean / Courier)
- Updated bottom quick-action strip using your provided icon images
- Mission & Vision section redesigned into branded cards

## Open locally
Option A: double-click `index.html`

Option B (recommended): run a local server
- VS Code → Extensions → "Live Server"
- Right-click `index.html` → "Open with Live Server"
